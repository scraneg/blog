部署静态网页反思

发表于 2023-05-05 更新于 2023-05-06 本文字数： 407 阅读时长 ≈ 1 分钟

这个博客其实在两年前就部署好了…犹记当时是web编程为我打开了新世界的大门，第一次在csdn上发表博客、第一次接触Web应用、第一次使用树莓派…虽然现在看来都已经很浅显了。

因为不满足csdn的局限性，所以当时很想要自己也有一个博客网页。这个过程中遇到最棘手的问题是设置SSH 密钥，不知道出了什么问题，找了助教老师也没有解决。最后发现其实很简单，本地的ssh key原本放到了github上，莫名其妙把本地的key删掉了新生成了一个，导致本地和云上的ssh不匹配，自然部署不了。以及图床有保护措施，博客里的图片可能会不显示，解决方法是给图床加白名单即可。其他问题因为时间太过久远都忘记了，不过印象里不算难。

但是博客部署好后并没有真正的使用起来，过了那一段比较有探索欲的时间马上就闲置了。（感觉后来写的文档都是学术垃圾，不值得放到博客上，也没什么参考价值…）

现在重新部署了一下，希望能把它建设起来，顺便多写点有用的文档…

websocket

WebSocket 是 HTML5 开始提供的一种在单个 TCP 连接上进行全双工通讯的协议。

WebSocket 使得客户端和服务器之间的数据交换变得更加简单，允许服务端主动向客户端推送数据。在 WebSocket API 中，浏览器和服务器只需要完成一次握手，两者之间就直接可以创建持久性的连接，并进行双向数据传输。

HTML5 定义的 WebSocket 协议，能更好的节省服务器资源和带宽，并且能够更实时地进行通讯。
